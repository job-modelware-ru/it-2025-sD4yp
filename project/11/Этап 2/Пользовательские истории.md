# Пользовательские истории веб-платформы для мониторинга арбитража ставок финансирования на BingX

## Роли и их заинтересованность:

### Гость
- **Кто**: человек, не имеющий аккаунта на платформе
- **Заинтересованность**: узнать больше о платформе, увидеть её приемущества, зарегистрироваться на платформе (после регистрации становится Трейдером)

### Трейдер (Основной пользователь)
- **Кто**: Криптотрейдер, использующий платформу для поиска и анализа арбитражных возможностей
- **Заинтересованность**: Максимизировать доходность от арбитража ставок финансирования при минимальных рисках и времени на анализ, получать своевременные уведомления о выгодных возможностях, иметь инструменты для глубокого анализа и расчета позиций

### Система (Веб-платформа)
- **Кто**: Автоматизированная система сбора, обработки и визуализации данных
- **Заинтересованность**: Обеспечивать точность и актуальность данных в реальном времени, предоставлять интуитивно понятный интерфейс, эффективно обрабатывать запросы к API биржи, обеспечивать надежность и производительность

### Администратор платформы
- **Кто**: Технический специалист, отвечающий за поддержку и развитие платформы
- **Заинтересованность**: Мониторить работоспособность системы, управлять доступом к API биржи, обновлять алгоритмы расчетов, обеспечивать безопасность данных пользователей
