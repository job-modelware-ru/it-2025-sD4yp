# Словарь терминов

- **Пользователь (Трейдер)**: Актор, использующий платформу для поиска, анализа и отслеживания арбитражных возможностей между рынками BingX.
- **Система**: Веб-приложение для мониторинга, анализа и визуализации данных по арбитражу ставок финансирования.
- **Биржа BingX**: Криптовалютная торговая платформа, предоставляющая спотовые и бессрочные фьючерсные рынки, а также API для получения рыночных данных.
- **Спотовый рынок (Spot Market)**: Рынок, на котором криптовалюты покупаются и продаются по текущей рыночной цене с немедленным расчетом.
- **Бессрочный фьючерс (Perpetual Futures)**: Деривативный инструмент, не имеющий даты экспирации, цена которого привязана к спотовому рынку и регулярно корректируется через ставки финансирования.
- **Ставка финансирования (Funding Rate)**: Периодический платеж между трейдерами на бессрочных фьючерсах, призванный сбалансировать цену фьючерса и спота. Может быть положительной или отрицательной.
- **Арбитраж**: Торговая стратегия, основанная на одновременной покупке и продаже одного актива на разных рынках для получения прибыли за счет разницы цен.
- **Арбитраж ставок финансирования (Funding Rate Arbitrage)**: Стратегия, при которой трейдер открывает противоположные позиции на споте и бессрочном фьючерсе, получая доход от ставки финансирования независимо от движения рынка.
- **Спред**: Разница между ценой актива на спотовом и фьючерсном рынках. Ключевой показатель для оценки арбитражных возможностей.
- **Годовая доходность (APY, Annual Percentage Yield)**: Прогнозируемая годовая доходность от арбитража, учитывающая ставки финансирования и комиссии.
- **Комиссия (Fee)**: Плата за совершение торговых операций (открытие, закрытие позиции, вывод средств), взимаемая биржей.
- **Маржа (Margin)**: Залог, необходимый для открытия позиции с кредитным плечом на фьючерсном рынке.
- **Кредитное плечо (Leverage)**: Механизм, позволяющий трейдеру открывать позиции, превышающие его собственный капитал, за счет заемных средств биржи.
- **Ликвидация (Liquidation)**: Принудительное закрытие позиции биржей при недостатке маржи вследствие неблагоприятного движения цены.
- **Портфель арбитражных позиций**: Совокупность открытых и закрытых арбитражных сделок пользователя, отслеживаемых на платформе.
- **Дашборд**: Главная страница платформы с ключевыми метриками, графиками и списком арбитражных возможностей.
- **Алерт (Alert)**: Автоматическое уведомление пользователя о появлении арбитражной возможности, превышающей заданные параметры доходности или ставки финансирования.
- **Фильтр**: Инструмент отбора торговых пар и арбитражных возможностей по заданным критериям (доходность, объем, волатильность).
- **Исторические данные**: Архив ставок финансирования, цен, спредов и других метрик за выбранный период для анализа стабильности стратегии.
- **График**: Визуальное представление динамики ставок финансирования, спредов, доходности и других показателей.
- **Калькулятор арбитража**: Инструмент для расчета потенциальной прибыли, рисков и необходимых параметров сделки с учетом капитала, плеча и комиссий.
- **API (Application Programming Interface)**: Программный интерфейс для автоматического получения рыночных данных с биржи BingX.
- **Веб-уведомление (Web Notification)**: Сообщение, отображаемое в браузере пользователя при срабатывании алерта или изменении статуса позиции.
- **Экспорт данных**: Функция выгрузки рыночных и пользовательских данных (история, портфель, ставки) в файл для внешнего анализа (CSV, Excel, JSON).
- **Сессия пользователя**: Временное состояние авторизации пользователя на платформе, позволяющее сохранять настройки и историю действий.
- **Регистрационная форма**: Веб-форма для создания учетной записи пользователя на платформе.
- **Личный кабинет**: Раздел платформы с настройками профиля, историей позиций, алертами и экспортом данных.
- **Роль пользователя**: Статус пользователя на платформе (гость, зарегистрированный трейдер, администратор).
- **Администратор**: Сотрудник, управляющий техническими и контентными аспектами платформы, включая модерацию, обновление данных и поддержку пользователей.
- **Волатильность**: Статистическая мера изменчивости цены актива, важная для оценки рисков арбитражных стратегий.
- **Торговая пара**: Комбинация двух криптовалют, между которыми осуществляется торговля (например, BTC/USDT).
- **Объем торгов**: Количество актива, купленного или проданного за определенный период, индикатор ликвидности рынка.
- **Период удержания позиции**: Время, в течение которого трейдер планирует держать арбитражную позицию для получения дохода.
- **Пользовательский пресет**: Сохраненный набор фильтров и настроек для быстрого доступа к интересующим торговым парам и стратегиям.
- **Симуляция P&L (Profit and Loss)**: Моделирование потенциальной прибыли и убытков при различных сценариях движения рынка.
- **Нереализованный P&L**: Текущая прибыль или убыток по открытой позиции, еще не зафиксированные трейдером.
- **Завершенная позиция**: Арбитражная сделка, закрытая пользователем, с итоговым расчетом прибыли и убытков.
- **История алертов**: Журнал всех сработавших уведомлений пользователя с деталями условий и времени.
- **Время до следующего платежа**: Таймер, показывающий, когда произойдет следующий расчет ставки финансирования на бессрочном фьючерсе.
- **Сортировка**: Организация списка арбитражных возможностей по выбранному критерию (доходность, объем, волатильность).
- **Адаптивный дизайн**: Техническая реализация интерфейса, обеспечивающая корректную работу платформы на различных устройствах (ПК, планшет, смартфон).
- **Интерактивный график**: График, позволяющий пользователю изменять масштаб, выбирать периоды и просматривать подробные значения метрик.
- **Обновление данных**: Автоматический или ручной процесс получения новых рыночных данных с биржи для актуализации информации на платформе.
