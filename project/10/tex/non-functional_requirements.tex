\section{Нефункциональные требования}

\subsection{Производительность}
\item Поддержка до \textbf{10 000 одновременных пользователей}.
    \item Время загрузки главной страницы — не более \textbf{2 секунд}.
    \item Генерация рекомендаций — не более \textbf{500 мс} на пользователя.

\subsection{Масштабируемость}
\item Архитектура должна поддерживать горизонтальное масштабирование.
    \item Возможность увеличения нагрузки в 10 раз без перестройки системы.

\subsection{Надёжность и доступность}
\item Доступность: \textbf{99.9\%} в год (не более 8.76 часов простоя).
    \item Ежедневное резервное копирование с возможностью восстановления за последние 7 дней.
    \item Отказоустойчивость ключевых сервисов.

\subsection{Безопасность}
\item Передача данных по \textbf{HTTPS (TLS 1.2+)}.
    \item Хранение паролей — хеширование (bcrypt/scrypt).
    \item Защита от XSS, CSRF, SQL-инъекций.
    \item Аудит безопасности — раз в квартал.
    \item Поддержка двухфакторной аутентификации (2FA) для разработчиков и менеджеров.

\subsection{Совместимость}
\item Поддержка Chrome, Firefox, Safari, Edge (2 последние версии).
    \item Адаптивный дизайн (от 320px).
    \item Интеграция с OAuth (Google, Apple, Discord, Steam).

\subsection{Удобство использования (Usability)}
\item Интуитивный интерфейс с чёткой навигацией.
    \item Поддержка тёмной и светлой темы.
    \item Локализация: русский, английский (дальнейшее расширение).

\subsection{Производительность алгоритма}
\item Гибридный подход: коллаборативная фильтрация + анализ контента + поведенческие сигналы.
    \item Обновление профиля пользователя — в течение \textbf{1 минуты}.
    \item Поддержка A/B-тестирования разных версий алгоритма.

\subsection{Данные и конфиденциальность}
\item Соблюдение GDPR и аналогичных норм.
    \item Пользователь может экспортировать или удалить свои данные.
    \item Минимальный сбор персональных данных с согласия.

\subsection{Мониторинг и логирование}
\item Аудит действий разработчиков и менеджеров.
    \item Мониторинг через Prometheus, Grafana, ELK.
    \item Оповещения о сбоях (Slack, email, SMS).

\subsection{Поддержка и сопровождение}
\item Панель управления для технической команды.
    \item Документация API.
    \item Тикет-система или чат-поддержка.
