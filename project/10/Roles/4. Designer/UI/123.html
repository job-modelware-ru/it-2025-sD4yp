PlayNext: Документация UI и Структура

Техническое описание интерфейса и декомпозиция компонентов для проекта PlayNext

1. Атомарные компоненты (UI Kit)

1.1 Button (Кнопка)

Основной элемент для совершения действий. Стиль: прямоугольная форма, сплошная заливка, тень.

Варианты и Состояния:
ТипПримерОписание
Primary"Change", "Yes"Основное действие, выделяется цветом (акцент).
Secondary/Default"Cancel", "No"Второстепенное действие, светлый фон/обводка.
Disabled"Disabled Button"Неактивна, не реагирует на нажатие.
1.2 Input Fields & Checkbox

Элементы для ввода данных и выбора опций. Используется `TextArea` для отзывов.

КомпонентТипВалидация (Форма регистрации)
InputEmailСтандартный формат email (пр. st.zinkin@yandex.ru).
InputPasswordМинимум 8 символов.
CheckboxChoiceДля выбора жанров и принятия условий.
1.3 Progress Bar (Индикатор)

Визуализация выполнения задачи или достижения прогресса.

Прогресс завершения профиля: 75%

Используется для отображения "Profile Completion" (75%) или "Achievement Progress".

2. Структурная декомпозиция (WBS - Визуальное дерево)

Иерархическое представление сложных UI-блоков, построенных из атомарных компонентов (WBS-стиль).

  • Организм: Форма регистрации (RegistrationForm) Сложный компонент для сбора данных, может быть в Модальном окне.
    • Молекула: Поля ввода (Input Group)
      • Атом: InputEmail, Password, Password Repeat (мин. 8 символов)
    • Молекула: Управление (Controls)
      • Атом: Checkbox"Accept terms and conditions"
      • Атом: Button"Register"
  • Страница: Профиль (ProfilePage) Отображает данные и настройки пользователя.
    • Блок: User Info & Stats
      • Атом: Stat ItemsRated games: 27, Reviews written: 8
    • Блок: Account Settings
      • Атом: Button"Change password", "Delete account"
  • Модальное окно: Предупреждение (Warning Modal) Универсальный элемент для подтверждения действия.
    • Текст: "Warning", "Are you sure?"
    • Молекула: Группа кнопок
      • Атом: Button"No" (Secondary)
      • Атом: Button"Yes" (Primary)
